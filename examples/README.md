# @aerostack/react-native Examples

Examples for `@aerostack/react-native`.

## Prerequisites

```bash
npm install @aerostack/react-native
```

> **Note**: This SDK is designed for React Native environments and uses native primitives where applicable.

## Available Examples

| Example | Description |
|---------|-------------|
| [**Basic Auth**](./basic-auth.tsx) | Login screen with `TextInput` and `Button`. |

## Usage

Copy the example code into your React Native or Expo project screen.
